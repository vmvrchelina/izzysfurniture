# Izzy's Furniture

Open the plugin with `/izf`.

Izzy's Furniture is a Dalamud plugin for building local scenes in Final Fantasy XIV with housing furniture, map assets, visual effects, customizable NPCs, and interior fixtures. This plugin creates client-only models, allows you to share your maps, and is persistent through G-Pose and live gameplay. 

This might crash you.
